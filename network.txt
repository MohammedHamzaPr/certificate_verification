========================
صح أو خطأ
========================

41. الـ Hub يرسل البيانات إلى جميع المنافذ. (   True   )
42. الـ Switch يرسل البيانات إلى الجهاز المقصود فقط. (   True   )
43. الـ Router يعمل داخل شبكة واحدة فقط. (   False   )
44. DNS يحول اسم الموقع إلى عنوان IP. (   True   )
45. كل جهاز في الشبكة يجب أن يمتلك عنوان IP مختلفًا. (   True   )

========================
اختيار من متعدد
========================

46. أي جهاز يربط بين شبكتين مختلفتين؟
أ) Hub
ب) Switch
ج) Router : True
د) Access Point

47. أي جهاز يرسل البيانات إلى جميع الأجهزة؟
أ) Router
ب) Hub : True
ج) Switch
د) Modem

48. DNS مسؤول عن:
أ) تشفير البيانات
ب) تحويل أسماء المواقع إلى عناوين IP : True
ج) إرسال البريد الإلكتروني
د) نقل الملفات

49. أي نوع شبكة يستخدم داخل منزل أو شركة؟
أ) WAN
ب) MAN
ج) LAN : True
د) PAN

50. أي بروتوكول يستخدم لتصفح المواقع بشكل آمن؟
أ) FTP
ب) HTTP
ج) HTTPS : True
د) SMTP
